# PAWSH

## Description

Pawsh is a copy-cat of Airbnb enabling animal parents to buy and sell pet fashion.

https://pawsh.herokuapp.com/

![Pawch landing page](/app/assets/images/Pawsh_landing.png?raw=true)

## Usage

Installation
```bash
bundle install
yarn install
rails db:migrate
rails db:seed:api_seeds
```
Run the app
```bash
yarn build --watch
```
